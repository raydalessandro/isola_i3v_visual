### 🎯 Checklist

**Casetta (signature):**
- [ ] Forma TONDA (circular floorplan, conical/domed roof)
- [ ] Modesta single-story
- [ ] Pareti ocra hand-applied
- [ ] Tetto terracotta scura o thatch (NON spiovente alto)
- [ ] Porta in legno semplice
- [ ] 1-2 finestre piccole con imposte
- [ ] Camino piccolo

**Cespuglio (signature CRITICA):**
- [ ] CESPUGLIO GRANDE che parzialmente nasconde la casa dal lato Piazza
- [ ] Foglie deep green (estate) o rami nudi (inverno)
- [ ] Senza il cespuglio = NON è la casa di Mèmolo

**Cortile:**
- [ ] Piccolo cortile modesto al lato
- [ ] Packed earth ground
- [ ] Eventuale panca + vaso di erbe
- [ ] NO decorazioni

**Contesto:**
- [ ] Hint Piazza dietro il cespuglio
- [ ] Hint Albero Vecchio chioma in distanza

**Anti-cliché:**
- [ ] NO personaggi, NO Mèmolo
- [ ] NO scritte / nastri / decorazioni
- [ ] NO Hobbit-style fairy-tale burrow
- [ ] NO mushroom roof
- [ ] NO bright painted shutters
- [ ] NO finestre rotonde Hobbit

> this is not Mèmolo's house.

#### Se rende fairy-tale (Hobbit/funghi/curve estreme):

> industrial. NOT a Hobbit burrow with mushroom roof, NOT a
